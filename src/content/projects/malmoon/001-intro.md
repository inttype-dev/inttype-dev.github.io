## 프로젝트 개요

**말문(MalMoon)** 은 WebRTC 기반 실시간 비대면 언어치료 서비스입니다. 재활사와 치료 대상자를 연결하고, AI와 디지털 교구를 통해 실제 치료실에 가지 않아도 언어재활을 받을 수 있는 환경을 만드는 것이 목표입니다.

SSAFY 13기 2학기 공통 프로젝트로, 약 6주간(2025.07.07 ~ 2025.08.18) 개발했습니다.

## 개발 배경

언어치료가 필요한 아이들은 생각보다 많지만, 실제로 치료를 받기까지의 장벽도 높습니다.

- 가까운 곳에 언어재활센터가 없는 지역 간 인프라 격차
- 높은 치료 비용
- 치료 대기 중 골든타임을 놓치는 문제

결국 치료가 필요한 시기에 접근조차 못하는 아이들이 생깁니다. 말문은 이 구조적인 문제를 해결하고자 기획했습니다.

**"누구든, 언제 어디서든 말의 꽃을 피울 수 있도록"** — 온라인으로도 실제 치료에 가까운 경험을 제공하는 것, 그게 말문의 출발점이었습니다.

## 핵심 기능

### 역할 기반 회원 관리

재활사와 치료 대상자로 회원 유형이 구분됩니다. 재활사는 자격증 번호와 경력을 입력해 전문성을 검증받고, 대상자는 기본 정보를 등록해 맞춤형 서비스를 받을 수 있습니다.

### 간이언어평가

대상자는 간단한 문항 기반 평가를 통해 현재 언어 능력을 진단받을 수 있으며, 평가 결과는 재활사 매칭 신청 시 함께 전달됩니다.

### 치료 매칭 및 스케줄 관리

재활사의 경력과 가능 시간대를 확인한 뒤 치료를 신청할 수 있습니다. 재활사는 본인 일정을 등록하고, 신청 온 대상자 정보와 진단 내역을 확인해 매칭을 관리합니다.

### WebRTC 기반 실시간 수업

화상으로 재활사와 대상자가 상호작용하는 치료 화면을 제공합니다. AAC 카드, 동화 교구 등 다양한 디지털 교구가 수업 화면에 통합되어 있습니다.

### AAC (보완대체의사소통)

발화가 어려운 대상자를 위해 상황·감정·행동을 표현할 수 있는 AAC 카드를 수업 중 활용할 수 있습니다. 재활사는 AI 이미지 생성으로 맞춤 AAC 카드를 직접 제작할 수 있습니다.

### 동화 STT 및 발화 평가

대상자가 동화를 읽으면 Whisper 기반 STT로 음성을 텍스트로 변환하고, 발화 정확도와 유창성을 분석해 평가 결과를 제공합니다.

### 집중력 필터

수업 화면에 모자, 동물 캐릭터 등 시각적 필터를 적용해 아이들의 집중도와 치료 참여도를 높입니다.

## 기술 스택 및 선택 이유

| 기술 | 선택 이유 |
|------|-----------|
| React + TypeScript | 컴포넌트 기반 UI 개발과 타입 안전성 확보 |
| Java + Spring Boot | 안정적인 백엔드 구조, 재활사/대상자 역할 분리 처리 |
| PostgreSQL | 매칭, 스케줄, 진단 데이터 등 복잡한 관계형 데이터 관리 |
| Python + FastAPI | STT, AAC 이미지 생성 등 AI 기능을 별도 서비스로 분리 |
| Whisper | 오픈소스 STT 모델로 발화 음성을 텍스트로 변환 |
| LiveKit / OpenVidu | WebRTC 기반 실시간 화상 통신 구현 |
| Redis | 화상 세션 방 정보 및 채팅 메시지 임시 저장, 방 삭제 재시도 큐 |

## 아키텍처

![MalMoon 아키텍처](/images/projects/malmoon/architecture_malmoon.png)

애플리케이션은 세 개의 레이어로 구성됩니다.

- **Frontend** — React 기반 SPA
- **Backend** — Spring Boot REST API 서버 (PostgreSQL, Redis 사용)
- **AI Service** — FastAPI 기반 서버 (STT, AAC 이미지 생성 처리)

실시간 화상 통신은 LiveKit/OpenVidu WebRTC 서버를 통해 처리되며, 파일 저장은 AWS S3를 사용합니다. 전체 서비스는 Docker로 컨테이너화되어 AWS EC2 위에서 운영됩니다.
