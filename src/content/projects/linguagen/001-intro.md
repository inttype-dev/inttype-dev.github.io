## 프로젝트 개요

**LinguaGen**은 생성형 AI와 게이미피케이션을 결합한 영어 학습 플랫폼입니다. ChatGPT가 사용자 수준에 맞는 문제를 자동 생성하고, 게임 형식의 학습과 포인트·랭킹 시스템으로 꾸준한 학습 동기를 제공합니다.

4인 팀(프론트엔드 1명, 백엔드 3명)이 약 2개월간(2024.10 ~ 2024.11) 개발했습니다.

## 개발 배경

영어 학습을 시작하는 사람은 많지만, 꾸준히 이어가는 사람은 적습니다.

- 단조로운 반복 학습으로 인한 흥미 저하
- 개인 수준을 고려하지 않은 획일적인 콘텐츠
- 동기 부여 부족으로 인한 낮은 지속률

이를 해결하기 위해 두 가지 방향을 잡았습니다.

**1. 게임 같은 학습 경험**

기존 영어 학습 플랫폼보다 훨씬 게임에 가까운 인터페이스를 만드는 것이 첫 번째 방향이었습니다. 2D 도트 그래픽 UI와 던전·산·폐허 맵 기반의 학습, 등급 업·업적·배지 시스템으로 학습 몰입도를 높였습니다. 등급은 유럽 연합의 CEFR(유럽공통언어평가기준)을 기반으로 브론즈부터 챌린저까지 6단계로 세분화했습니다.

**2. 관심사 기반 맞춤형 문제 생성**

사용자가 관심사를 입력하면 ChatGPT가 그에 맞는 문제를 자동으로 생성합니다. 관심사 7종류, 문제 유형 3종류, 사용자 등급을 조합해 다양한 문제를 제공하며, 목록에 없는 관심사도 직접 입력해 문제를 만들 수 있습니다.


## 핵심 기능

### AI 문제 생성

ChatGPT API를 기반으로 주제, 등급(브론즈~챌린저), 문제 유형(리딩/리스닝/기타)을 선택하면 그에 맞는 영어 문제를 자동 생성합니다. 사용자가 직접 문제 세트를 구성해 학습할 수 있습니다.

### 게임 기반 학습

던전, 산, 폐허 등의 맵에서 영어 문제를 풀며 진행하는 게임 형식의 학습을 제공합니다. Three.js 3D 씬과 Lottie 정답/오답 애니메이션으로 몰입감을 높였으며, 클리어 시 포인트가 적립됩니다.

### 레벨 테스트 및 맞춤 학습

브론즈부터 챌린저까지 6단계 등급을 진단하는 레벨 테스트를 제공합니다. 테스트 결과는 AI 문제 생성과 학습 분석에 활용됩니다.

### 데일리 퀴즈 / 오늘의 단어

매일 새롭게 제공되는 퀴즈와 단어로 학습 습관을 형성할 수 있습니다. 완료 시 포인트가 지급됩니다.

### 피로도 시스템

무제한 학습을 방지하고 적정 학습량을 유도하는 피로도 시스템을 도입했습니다. 메인 페이지에서 피로도 현황을 실시간으로 확인할 수 있습니다.

### 포인트 / 랭킹

문제 풀이, 게임 클리어, 데일리 퀴즈 등 다양한 활동으로 포인트를 획득하고, 주간 등급별·전체 랭킹에서 순위를 확인할 수 있습니다.

### 대시보드 및 커뮤니티

학습 통계, 배지 획득 내역, 취약 유형 분석을 대시보드에서 시각적으로 확인할 수 있습니다. 커뮤니티에서는 자유게시판, 학습 팁 게시판 등을 통해 다른 학습자와 소통할 수 있습니다.

## 기술 스택 및 선택 이유

| 기술 | 선택 이유 |
|------|-----------|
| React + JavaScript | 컴포넌트 기반 SPA 개발 |
| Three.js | 게임 학습 맵의 3D 씬 구현 |
| Java + Spring Boot | 안정적인 백엔드 구조, JPA 기반 도메인 설계 |
| OpenAI Java 라이브러리 | Spring Boot에서 직접 GPT API를 호출해 문제 생성 |
| MySQL | 문제, 학습 기록, 랭킹 등 복잡한 관계형 데이터 관리 |

## 아키텍처

![LinguaGen 아키텍처](/images/projects/linguagen/architecture_linguagen.png)

프론트엔드(React)와 백엔드(Spring Boot) 두 레이어로 구성됩니다. Nginx가 리버스 프록시 역할을 하며 정적 파일 서빙과 API 요청 분기를 담당합니다. AI 문제 생성은 별도 서버 없이 Spring Boot에서 OpenAI API를 직접 호출합니다.
