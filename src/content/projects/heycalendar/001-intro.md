## 프로젝트 개요

**HeyCalendar**는 학생의 장학금 탐색·신청과 관리자의 심사·운영을 하나의 플랫폼에서 처리하는 원스톱 장학금 통합 관리 서비스입니다.

2025 신한 해커톤 with SSAFY에서 4인 팀 **쏠SOL이들**이 3일간(2025.08.29 ~ 2025.08.31) 개발했습니다.

## 개발 배경

장학금은 학생들에게 중요한 혜택이지만, 막상 찾아서 신청하기까지의 과정은 쉽지 않습니다.

- 장학금 정보가 여러 곳에 흩어져 있어 한눈에 파악하기 어려움
- 신청 절차가 복잡하고 문서 제출 과정이 불편함
- 관리자는 학생별 지원 현황을 실시간으로 파악하기 어려움

HeyCalendar는 그 두 가지 불편함을 하나의 플랫폼에서 해결하는 것을 목표로 했습니다.

## 핵심 기능

### 사용자 앱 (모바일)

학생은 React Native 기반 모바일 앱을 통해 장학금을 검색하고 신청할 수 있습니다. 문서는 앱에서 직접 업로드하고, 모집 일정과 심사 결과는 알림으로 받아볼 수 있습니다.

### 관리자 웹

관리자는 별도의 웹 어드민 페이지에서 장학금 공고를 등록·수정하고, 학생들의 지원 현황을 실시간으로 관리합니다. 심사 및 승인 처리도 웹에서 진행합니다.

## 기술 스택 및 선택 이유

| 기술 | 선택 이유 |
|------|-----------|
| React Native + Expo | 단일 코드베이스로 모바일 앱 개발, 빠른 프로토타이핑에 유리 |
| React + Vite | 관리자 웹을 빠르게 구성하기 위한 SPA 개발 |
| Java + Spring Boot | 안정적인 백엔드 REST API 서버 구성 |
| MySQL | 관계형 데이터 저장 (AWS RDS) |

## 아키텍처

사용자 앱(React Native)과 관리자 웹(React)이 각각 Spring Boot REST API 서버와 통신하는 구조입니다. DB는 AWS RDS(MySQL)를 사용하고, 파일은 S3에 저장합니다. 전체 서비스는 Docker로 컨테이너화되어 AWS EC2 위에서 Nginx 리버스 프록시를 통해 운영됩니다.
